# TELIPORT AI Autism Platform

A full-stack modern web app: dark futuristic medical AI dashboard with React (Vite), Tailwind CSS, Framer Motion, and Node.js + Express.

## Structure

- **frontend/** — React + Vite, Tailwind, Framer Motion, React Router
- **backend/** — Node.js + Express API

## Quick start

### Backend

```bash
cd backend
npm install
npm run dev
```

Runs at `http://localhost:3001`.

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs at `http://localhost:5173`. API requests to `/api/*` are proxied to the backend.

## Tech stack

- **Frontend:** React 18, Vite, Tailwind CSS, Framer Motion, React Router
- **Backend:** Node.js, Express, CORS
- **Theme:** Dark medical AI dashboard, neon green + blue accents, glassmorphism, soft glow

## Routes

- `/` — Dashboard
- `/assessments` — Assessments
- `/sessions` — Sessions
- `/settings` — Settings
